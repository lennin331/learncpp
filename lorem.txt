To paste the entire content of a file in Vim, use the :%y command to yank (copy) all lines in the file into the default register, then paste it elsewhere. 

Steps to Copy and Paste the Entire File
Ensure you're in Normal mode (press Esc if needed). 
Yank the entire file:
Type :%y and press Enter.
% represents all lines from line 1 to the last line (1,$).
This copies the entire file to the default register.
Move to the desired location (e.g., another file, buffer, or new location in the same file). 
Paste the content:
Press p to paste after the cursor.
Press P to paste before the cursor.
